# "Hello World" to readers! 👋

## About me

I'm Jacob, a 20-year-old CNC Operator with a passion for programming and a goal to break into IT.

My journey started at 17 when I discovered Lua while playing MTA:SA. Lua wasn't for me, but it sparked my curiosity. When I moved to GTA:V multiplayer platforms, I wanted to build my own mods and gamemodes — that's when I found JavaScript.

Back then I didn't take it seriously. I wrote code just to make things work, without caring about performance, clean code, or SOLID principles — I hadn't even heard of them yet.

Then AI became popular. I made a classic mistake — I used it as a code machine: "write me this", copy, paste. It wasn't learning, it was an illusion of learning. I noticed my thinking was degrading, I couldn't solve problems on my own anymore.

After a 2 months of searching for the cause, I found it. I went on an AI detox and Social media detox — forcing my brain to find solutions independently. My memory improved, my problem-solving skills reached an ATH.

Now I'm fully focused on TypeScript — my language of choice. I'm self-taught, building real projects, and working toward my goal every day.
